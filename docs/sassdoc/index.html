    variables

    blue-grid-options-base

    $blue-grid-options-base: (
  columns: 12,
  gutter-width: 2.5%,
  container-class: 'cs',
  column-class: 'cols',
  breakpoints: (
    s: 480px,
    m: 600px,
    l: 768px,
    xl: 1024px,
    xxl: 1200px
  ),
  output-sizes: (
    m: (4, 6, 8),
    xl: (3, 4, 6, 8, 9)
  )
) !default;

    Description

    Default grid options. The options you pass into the blue-grid mixin get merged into this map to determine your grid config.

    Type

    Map

    Map structure

    Map keyNameMap keyDescriptionMap keyTypeMap keyValue
    columns

    The number of columns in the grid system

    		Number12
    gutter-width

    A single gutter width in %

    		Percentage2.%
    container-class

    Class for the grid container

    		String'cs'
    column-class

    Class prefix for the grid columns

    		String'cols'
    breakpoints

    Map of breakpoint names and sizes

    		Map—none
    output-sizes

    Map of breakpoint names to column counts (so that we don't just spit out every possible nested combination of columns)

    		Map—none
